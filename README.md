This project is a diploma work that represents an original first-person retro shooter, inspired by classic games like DOOM (1993).

The player takes control of a revived zombie, fighting against police and military forces while progressing through enclosed levels and surviving enemy encounters.

The game is built in a 2.5D pixel-art style, where 3D environments are visually designed to resemble classic sprite-based graphics.

⚙️ Technologies Used
Unity (Game Engine)
C#
TextMeshPro (UI)
Aseprite (pixel art)
Blender (3D models & environment)
🎯 Core Features
🔫 Shooting System (Hitscan)
Instant hit detection using Raycast
Visual effects: blood and bullet holes
Fire rate limitation
Sound effects and shooting animations

❤️ Health & Armor System
Damage is first absorbed by armor
Remaining damage is applied to health
Max value limitations
UI updates via event-driven system

📦 Pickup System
Ammo
Armor
Medkits (only usable when health is not full)

👾 Enemy AI
Player detection
Pursuit behavior
Ranged attack system
Line-of-sight check using Raycast
Death animations

🧠 Architecture
IDamageable interface for damage handling
Separation of responsibilities (modular scripts)
Event-based UI updates (optimized performance)

🕹️ Controls
WASD — Movement
Mouse — Look around
Left Mouse Button — Shoot
ESC — Pause menu

🧩 Project Highlights
Retro / pixel-art visual style

🗺️ Level Design
Handcrafted level (no procedural generation)
